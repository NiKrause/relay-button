# Docusaurus App

This directory now contains the Docusaurus app scaffold for
`relay-button`.

What is in place:

- the site `package.json`
- `docusaurus.config.js`
- `sidebars.js`
- custom CSS
- a small landing page
- the current implementation-backed docs content under `docs/`

What is still not done:

- dependencies have not been installed in this workspace yet
- the site has not been built or previewed yet in this environment

Once dependencies are installed, the repo-level scripts in `package.json`
can be used:

- `pnpm docs:dev`
- `pnpm docs:build`
